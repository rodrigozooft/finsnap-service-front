# FinSnap Connect Frontend

## Project Context
- finsnap-services: Backend API repository (MCP server available)
- finsnap-website: Marketing website (logo, colors, design reference)
- finsnap-service-front: This repository - Connect dashboard frontend

## Tech Stack
- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Zustand (state management)
- React Query (data fetching)
- React Router v7

## API Base URL
- Development: `/api/v1` (proxied to `http://localhost:8000`)

## Design System
- Primary color: #5171ff
- Font: Montserrat
